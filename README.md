# microlib
Minimal C++ utility library for 2D games, for Mac OSX, Windows, and now Raspberry Pi.

Features:
 * No graphics!
 * Game class
 * Game state base class
 * Game object base class
 * Message queue
 * Resource manager
 * Timer class
 * File IO helper classes.

To build the library:

  `make`

To run the unit tests:

  `make runtests`
